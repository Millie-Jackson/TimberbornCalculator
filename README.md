A Python + Gradio colony planner for Timberborn 1.0.

## Goals
- Help players plan food, water, storage, power, wellbeing, and growth
- Support mixed populations: adults, kits, and bots
- Expand building dependencies and upstream needs
- Provide automation and problem-solving suggestions
- Host the tool publicly on Hugging Face Spaces

## Current scope
- Folktails first
- Gradio UI from the start
- Data-driven patch values for easier updates

## Planned features
- Colony overview
- Needs calculator
- Dependency planner
- Power planner
- Wellbeing suggestions
- Automation helper
- Problem -> solution advice
- Nerdy Mode
